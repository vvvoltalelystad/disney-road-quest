# Speciale prestatiebadges

De zes definitieve vierkante PNG-bestanden staan in deze map. De app gebruikt deze automatisch; als een bestand ontbreekt, wordt de ingebouwde luxe placeholder getoond.

- `special_common_complete.png` — De Eerste Vonk
- `special_uncommon_complete.png` — Ontdekkingsreiziger van Twee Werelden
- `special_rare_complete.png` — Hoeder van de Magie
- `special_epic_complete.png` — Meester van Dromen en Avontuur
- `special_legendary_complete.png` — Legende van de Twee Parken
- `special_master_collector.png` — Het Hart van Disneyland Paris

De voor mobiel geoptimaliseerde ontwerpen zijn 768 × 768 pixels, zonder tekst, op dezelfde diep-donkerblauwe juwelendoosachtergrond. Alle belangrijke vormen blijven binnen de middelste 80%, zodat ze ook op een klein telefoonscherm duidelijk zijn.

De gebruikte ontwerpprompts staan in `DESIGN_PROMPTS.md`.
